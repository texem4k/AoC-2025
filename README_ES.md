# Advent of Code 2025

### Bienvenido a mi proyecto del AoC 2025.
###
#### ¿Por qué estoy haciendo esto?

Esta idea surgió de mi profesor de Ingeniería del Software,
y la verdad es que me lo estoy pasando muy bien haciéndolo. Muy buena idea, profe.

Si has llegado aquí y no sabes qué es Advent of Code, básicamente consiste en
resolver un problema de programación cada día (con parte I y II) hasta el día del adviento.
Normalmente dura hasta el 24, pero este año llega hasta el 12.
###
#### ¿Cuál es el objetivo?

El objetivo principal es desarrollar buenos hábitos y mentalidad de código limpio:
usar los patrones, principios y buenas prácticas que hemos aprendido.
De paso, aplicamos conceptos que son realmente útiles en la vida real.
###
#### ¿Cómo está organizado el proyecto?

La estructura del repositorio es la siguiente:

* **Doc/** → Documentación. Aquí está la info de cada problema.

* **src/main/** → Código de producción, lo bueno de verdad :)

* **src/test/** → Entorno de testing. Una clase por problema, con sus pruebas correspondientes.

  * **src/test/resources/** → Inputs del AoC para cada reto diario.

Todo el código está documentado para facilitar su lectura y comprensión.
